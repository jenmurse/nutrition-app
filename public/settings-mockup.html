How to set up
1.
Configure your Assistant

Choose your preferred AI host and update its configuration file. We recommend Claude Desktop for complex meal planning or Roo Code/Windsurf for an integrated development experience.

Assistant Config File Path
Claude Desktop ~/Library/Application Support/Claude/claude_desktop_config.json
Cursor ~/.cursor/mcp.json
Windsurf / Roo Code ~/.codeium/windsurf/mcp_config.json

Copy and paste the block below into your config file. If you already have other servers configured, simply add the good-measure object to your existing mcpServers list.

JSON 
{
  "mcpServers": {
    "good-measure": {
      "command": "npx",
      "args": ["-y", "good-measure-mcp"],
      "env": {
        "GOOD_MEASURE_API_URL": "https://your-app.vercel.app",
        "GOOD_MEASURE_API_TOKEN": "YOUR_GENERATED_TOKEN"
      }
    }
  }
}

Pro Tip: No manual installation is required. npx will automatically fetch the latest version of the Good Measure server each time your assistant starts.

2.
Activate & Test

Restart your AI assistant to initialize the connection.

Try this prompt to test the link:

"List my recipes from Good Measure and tell me which ones are high in protein."

Optimization Workflow (Advanced)

To use the full power of the Chef Agent for meal prep, try a prompt like this:

"Get the recipe for Black Bean Avocado Brownies. Analyze it for nutritional optimization—check my database for substitutions using search_ingredients and show me a comparison table before saving."
